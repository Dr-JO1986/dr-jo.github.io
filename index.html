หน้าหลัก

สวัสดีครับ ผมพงษ์สวัสดิ์ เจ้าหน้าที่ธุรการจาก อบต.คำนาดี จังหวัดร้อยเอ็ด สนใจเทคโนโลยี การเขียนเว็บ และการให้บริการประชาชนผ่านดิจิทัล

บทความ

รวมบทความสั้น ๆ ที่เกี่ยวกับงานราชการ เทคโนโลยี และการใช้เครื่องมือดิจิทัลในการพัฒนาชุมชน

  • การใช้ Google Apps Script เบื้องต้น
  • จัดการเอกสารราชการออนไลน์อย่างมีประสิทธิภาพ
  • 3 เทคนิคสร้างเว็บแบบง่าย ๆ ด้วย GitHub Pages

ถาม-ตอบ

มีคำถามเกี่ยวกับเทคโนโลยี การใช้เครื่องมือราชการออนไลน์ หรือแนวทางการพัฒนาเว็บไซต์? ส่งคำถามมาได้เลยครับ

  • ❓ ถาม: ทำเว็บ GitHub Pages ต้องเสียเงินไหม?
    ✅ ตอบ: ไม่ต้องครับ ใช้ได้ฟรีตลอดชีวิต
  • ❓ ถาม: ต้องเรียนโค้ดเก่งไหม?
    ✅ ตอบ: แค่ HTML/CSS พื้นฐานก็พอ เริ่มจากไฟล์เดียวได้เลย
